Простой проект на базе FastAPI и Confluentic_Kafka для примера взаимодействия.

Для сборки в докере - представлен докер файл. Сборка стандартным методом.

Для запуска:
docker run -d -p 1234:8888 --network fastapiprobe_default test1 --broker-socket kafka1:9099 --broker-timeout 0.100
* - требуется включение в сеть кафки, т.к. она поднимается с помощью docker compose и работает в совей отдельной сетке докера.
* - два аргумента в конце - указание сокета (в формате, как указано в yaml файле для запуска кафки с помощью docker compose), и указание таймаута для считывания сообщений по ручку get.
